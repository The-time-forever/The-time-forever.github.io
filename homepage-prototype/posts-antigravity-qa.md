# 文章归档页原型 QA

## 核对对象

- 视觉原稿：`designs/posts-antigravity-desktop-v1.png`
- HTML 原型：`posts-antigravity.html`
- 桌面验证：1440 × 900
- 移动端验证：390 × 844

## 视觉一致性记录

| 核对项 | 原稿依据 | 实现结果 |
| --- | --- | --- |
| 首屏构图 | 左侧大标题，右侧留白，粒子横向散开 | 保留同一视觉重心和纯留白；不再放置粒子交互说明 |
| 字体层级 | 大号中文衬线标题、拉开字距的英文标题 | 使用仓库可直接运行的系统中文衬线字体栈，比例和节奏一致 |
| 色彩 | 米白纸色、黑色正文、青色强调 | 保持；另提供可记忆的深色模式 |
| 粒子 | 蓝青色短线和点状粒子，左侧更密 | Canvas 实现；指针产生排斥，离开后回归轨道 |
| 容器模型 | 开放式栅格与细分隔线，不使用卡片 | 保持，无卡片、无黑框、无玻璃面板 |
| 文章列表 | 纵向年份、两位序号、标题、日期和标签 | 使用仓库真实文章标题、日期和链接完成 |
| 移动端 | 原稿未提供 | 转为单列，保留年份轴；筛选横向滚动，无横向页面溢出 |

## 功能验证

- 搜索 “Agent” 后仅显示对应的 Agent 术语文章。
- 选择“技术折腾”后显示 6 篇文章。
- 深浅色切换可用，并写入本地存储。
- 粒子 Canvas 在桌面与移动端均渲染；鼠标和触摸指针均可扰动粒子。
- `prefers-reduced-motion` 下停止持续动画。
- 不蒜子统计使用原有 `busuanzi_container_*` 与 `busuanzi_value_*` 标识；原型不加载远程统计脚本，正式接入时沿用仓库现有脚本。
- 已按反馈移除首屏右侧“移动光标”说明及其分隔线，粒子交互仍正常工作。

## 有意差异

- 原稿中的文章标题和访问数字由生成模型虚构；HTML 全部替换为仓库真实文章数据，统计值用破折号占位。
- 原稿只表现一屏 6 篇文章；HTML 收录目前用于原型的 11 篇仓库文章，因此页面可继续滚动。
- 未复制 Google Antigravity 的品牌、导航、文案或专有粒子实现，只借鉴疏密关系和反重力交互概念。
